Uni-NaVid

Vicuna-7B · EVA-CLIP · RSS 2025 · arXiv 2412.06224

A video-based vision-language-action model for embodied navigation, built on a Vicuna-7B language model and an EVA-CLIP vision encoder.

Language modelVicuna-7B-v1.5
Vision encoderEVA-CLIP ViT-G

Mechanism

Uni-NaVid streams one egocentric RGB video into a single Vicuna-7B policy. To bound the cost of a long episode, it merges redundant long-term tokens when their cosine similarity exceeds a threshold τ = 0.95. History is held in this pooled form inside the model, so each step adds one frame instead of reprocessing the past.

Action output

Each inference emits the next four actions, from the discrete set {FORWARD, TURN-LEFT, TURN-RIGHT, STOP}. These actions correspond to fixed magnitudes: a 25 cm forward movement and a 30° turn.

Evaluation

On VLN-CE Val-Unseen the paper reports R2R success rate 47.0 and RxR success rate 48.7. On object-goal navigation it reports ObjectNav HM3D success rate 73.7, from the same checkpoint.
