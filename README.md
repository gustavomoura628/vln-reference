# VLN reference

A cited quick-reference comparing vision-language navigation (VLN) models — base model, input,
action output, mechanism, and official evaluation, on shared axes. **Every fact is footnoted to a
primary source**: hover a marker to see the exact location and a verbatim quote, with a link to
the paper or repo.

**Live:** https://gustavomoura628.github.io/vln-reference/

Zero-dependency static site (HTML/CSS/JS) — no build step. Open `index.html` directly, or host
anywhere. Citations link to arXiv and source repositories.
